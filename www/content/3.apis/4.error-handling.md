## Standard HTTP Response Codes

The following table lists the standard HTTP response codes:

| Code | Description |
|------|-------------|
| 202 (Accepted) | The request has been accepted for processing. |
| 400 (Bad Request) | The server could not process the request due to client error (malformed JSON, invalid field values, missing required parameters, or invalid query).  |
| 401 (Unauthorized) | Authentication is required or the provided credentials/token are missing or invalid.  |
| 50x (Internal Server Error) | An unexpected error occurred on the server.  |


## <span class="inline-block px-2 py-0.5 rounded bg-green-100 text-green-700 text-lg font-semibold font-mono font-bold">POST</span> `/v1/Inventory/Sync`

### Bad Request (400) Response

Missing required fields. The payload failed validation and the API returns `ValidationProblemDetails` with field-level errors.

::tabs{variant="line"}
  ::div{label="Sample Response"}
  ```json
  {
  "errors": {
    "FulfillmentCentreId": [
      "The FulfillmentCentreId field is required."
    ],
    "Provider": [
      "The Provider field is required."
    ]
  },
  "type": "https://httpstatuses.com/400",
  "title": "One or more validation errors occurred.",
  "status": 400,
  "detail": "Validation failed for one or more fields.",
  "traceId": "00-0403053dcf6dc794b02b09b0ca3ef956-e6eda743157a8d75-0"
  }
  ```
  ::
  ::div{label="Possible Workarounds"}
  - Required fields must be present.
  - For every item: `sku` length 3â€“32 and `articleStatus` must be present.
  - Check for malformed JSON request payload.
  - No empty strings for required fields.
  - No extra properties anywhere (strict schema).
  ::
::

### Not authenticated (401) Response

Authentication is required or the provided credentials/token are missing or invalid.

::tabs{variant="line"}
  ::div{label="Sample Response"}
  ```json
  {
  "errors": {
    "Authorization": [
      "Missing or invalid Bearer token."
    ]
  },
  "type": "https://httpstatuses.com/401",
  "title": "Unauthorized",
  "status": 401,
  "detail": "Not authenticated. Get a security token using ESW credentials and include Authorization: Bearer <token>.",
  "traceId": "00-5a1c3d2e4f6a7790bbccddeeff001122-abcdefabcdefabcd-0"
  }
  ```
  ::
  ::div{label="Possible Workarounds"}
  - Ensure the `Authorization` header is present with a valid API key or JWT token.
  - Get a security token using ESW credentials.
  ::
::

### Internal Server Error (500) Response

An unexpected error occurred on the server.

::tabs{variant="line"}
  ::div{label="Sample Response"}
  ```json
  {
  "errors": {
    "Server": [
      "An unexpected error occurred."
    ]
  },
  "type": "https://httpstatuses.com/500",
  "title": "Internal Server Error",
  "status": 500,
  "detail": "An unexpected error occurred on the server.",
  "traceId": "00-5a1c3d2e4f6a7790bbccddeeff001122-abcdefabcdefabcd-0"
  }
  ```
  ::
  ::div{label="Possible Workarounds"}
  - Retry the request after a brief wait.
  - Contact support with the traceId for further assistance.
  ::
::
