## Import Shopify Orders into ESW System

::steps
### Subscribe or Receive the Order Created Webhook
Import Shopify orders into ESW as offline orders.

::alert{title="Webhook" smart-icon="logos:shopify" size=40 to="https://shopify.dev/docs/api/admin-rest/2025-04/resources/webhook#event-topics-orders-create" target="_blank"}
  `orders/create`
::

curl -i -X GET \
